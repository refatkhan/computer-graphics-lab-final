# 🌳 Animated Natural Environment using OpenGL (GLUT)

## 📌 Project Overview
This project is a **2D animated park scene** developed using **OpenGL and GLUT**.  
It simulates a natural environment with multiple objects and dynamic animations to create a realistic and interactive graphical scene.

The scene includes elements like trees, a pond, a bench, a lamp post, clouds, and birds.  
It also supports environmental effects such as **day/night transition** and **rainfall simulation**.

---

## 🎯 Features
- 🌞 Day and 🌙 Night Mode Toggle  
- 🌧️ Rain ON/OFF Control  
- ☁️ Animated Clouds  
- 🐦 Flying Birds with Wing Animation  
- 🌊 Water Wave Animation in Pond  
- 💡 Lamp Lighting Effect at Night  
- 🌳 Realistic Trees and Environment  
- ⌨️ Keyboard Interaction Support  

---

## 🎮 Controls
| Key | Action |
|-----|--------|
| `D` | Switch to Day Mode |
| `N` | Switch to Night Mode |
| `R` | Toggle Rain |

---

## ⚙️ How It Works
The project uses a **rendering loop** where:
1. Objects are drawn using OpenGL primitives  
2. Animation variables are updated continuously  
3. The scene is redrawn using `glutPostRedisplay()`  

This creates smooth real-time animation.
